# Contributing

This is a personal solo project and is not currently open for contributions.

However, you are welcome to:

- **Report bugs** — open an issue if you find something broken
- **Suggest ideas** — open an issue to suggest a feature or improvement
- **Fork it** — feel free to fork and build your own version (non-commercial use only, per the license)

## If you do open an issue, please include:

- What you were trying to do
- What you expected to happen
- What actually happened
- Any error messages or screenshots
